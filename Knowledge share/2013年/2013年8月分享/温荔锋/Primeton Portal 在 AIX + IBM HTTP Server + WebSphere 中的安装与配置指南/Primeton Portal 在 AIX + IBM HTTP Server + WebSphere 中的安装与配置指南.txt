本文从 Primeton Portal 的配置集成出发，介绍 Primeton Portal 在 AIX 环境下，与 IBM HTTP Server7 + WebSphere 7 的相关配置过程。

与单点登录相关的内容请参考《SSO 集成指南》 文档。
